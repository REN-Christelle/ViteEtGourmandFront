# Vite & Gourmand
Vite & Gourmand est une entreprise en pleine expansion spécialisée en restauration événementielle à Bordeaux et qui souhaite développer un site web leur permettant d’augmenter leur visibilité ainsi que de proposer des menus plus facilement et plus accessibles à plus de monde.
Ce site est un site vitrine pour ce restaurant.

# Environnement de développement
- Système d'exploitation : Windows 11
- IDE : Utilisation de Visual Studio Code avec les extensions PHP Intelephense, Format HTML in PHP, Live Sass Compiler, PHP Server.
- Backend : PHP 8.3.30 / Symfony 6.4 / Symfony CLI
Lancement du serveur avec la commande suivante : symfony server:start
- Base de données : utilisation de MySQL, gestion avec PhpMyAdmin, et MongoDB pour la gestion des statistiques.
- Frontend : HTML, CSS Bootstrap installé via npm

# Installation du projet
Prérequis : PHP 8.3, SymfonyCLI, MySQL, Node.js & npm

Etapes d'installation :
1. Cloner le projet :
git clone https://github.com/REN-Christelle/ViteEtGourmandBack.git (pour le backend)
git clone https://github.com/REN-Christelle/ViteEtGourmandFront.git (pour le frontend)

2. Installer les dépendances :
- Installer Symfony CLI (https://scoop.sh/)
- Installer Bootstrap : npm install bootstrap

3. Lancer le serveur backend : symfony server:start

4. Lancer la page frontend : utilisation de PHP Server: Serve Project
